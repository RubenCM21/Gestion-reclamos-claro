Cada rol tiene una experiencia distinta, permisos específicos y módulos adaptados a sus responsabilidades.

👤 Personas

Cliente Persona

Para usuarios que desean registrar reclamos, reportar incidencias, consultar seguimiento y revisar notificaciones personales.

  • Registrar reclamos e incidencias
  • Consultar estado de casos
  • Adjuntar evidencias
  • Recibir notificaciones
🏢 Empresas

Cliente Empresa

Para representantes de empresas que gestionan tickets, servicios empresariales, incidencias técnicas, cloud, correo o conectividad.

  • Reportar tickets empresariales
  • Consultar SLA y prioridad
  • Gestionar servicios cloud
  • Revisar historial empresarial
🎧 Operativo

Asesor de Atención

Para trabajadores encargados de atender casos asignados, actualizar seguimiento, responder solicitudes y registrar acciones.

  • Ver bandeja asignada
  • Actualizar atención
  • Solicitar información
  • Registrar cierre o avance
📊 Gestión

Supervisor

Para personal encargado de clasificar casos, asignar asesores, monitorear SLA, revisar indicadores y controlar vencimientos.

  • Clasificar y priorizar casos
  • Asignar o reasignar responsables
  • Supervisar indicadores
  • Controlar SLA
⚙️ Sistema

Administrador

Para usuarios autorizados que administran parámetros, roles, usuarios, catálogos, auditoría, notificaciones e integraciones.

  • Gestionar usuarios y roles
  • Configurar catálogos
  • Administrar SLA
  • Revisar auditoría
